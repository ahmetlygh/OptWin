**OptWin** is a powerful, open-source web-based tool designed to generate custom Windows optimization batch scripts. Whether you're a gamer, a power user, or just want a cleaner system, OptWin lets you choose exactly what you want to optimize.

## üöÄ Features

### **System Optimization**
* **Clean Temporary Files**: Removes junk from `%temp%` and `Windows/Temp`.
* **Clean Prefetch**: Refreshes system caching.
* **Empty Recycle Bin**: Instantly clears deleted files.
* **Disable Hibernate**: Saves gigabytes of disk space (`hiberfil.sys`).
* **Performance Modes**: Switch between High Performance and **Ultimate Performance** plans.
* **Disable Telemetry**: Block Windows tracking and data collection.

### **Network & DNS**
* **Flush DNS**: Fix connection issues by resetting the cache.
* **Network Stack Reset**: Reset Winsock and TCP/IP.
* **Disable Throttling**: Remove Windows network throttling limits.
* **DNS Switcher**: Easily switch to Cloudflare, Google, OpenDNS, Quad9, AdGuard.
* **Ping Test Tool**: Built-in latency tester to find the fastest DNS for you.

### **Maintenance & Services**
* **DISM Health Check & Repair**: Fix corrupted Windows system images.
* **SFC Scan**: Repair broken system files.
* **Services Management**: Disable unused services like Fax, Wallet, Maps, Print Spooler, Touch Keyboard, and Xbox Services to free up resources.

### **Extra Tweaks**
* **Disable Bing Search**: Remove Bing results from the Start Menu.
* **Disable Sticky Keys**: No more annoying shift-key popups.
* **Show File Extensions**: See exactly what files you are running.
* **Disable Mouse Acceleration**: Get raw input for gaming.
